Convert font-encoded .docx to Unicode

This site offers conversions to Unicode text for some languages and encoding fonts.

Note that converting a file does not store the contents online.

Available conversions

  • Adlam: Aissata, Fuuta, Pulaar to Unicode
  • Latin Fulfulde: Latin text to Adlam Unicode
  • Ahom: AHOMFONT & Ahom Manuscript to Unicode
  • Tai Phake: ASCII encoding to Unicode

Note: Each paragraph is analyzed. If a paragraph's content is detected to be either English or French, that pararagraph is not converted.
